# react_reactrouter_redux_immutable_typescript
记录一次使用react全家桶搭配typescript开发的环境搭建
技术栈：
##### 1)react;
##### 2)react-router4;
##### 3)redux;
##### 4)redux-immutable;
##### 5)webpack;
##### 6)babel;
##### 7)eslint.


下载代码后执行
```$xslt
npm  install
```
安装完成后使用
```$xslt
npm run start
```
启动webpack-dev-server,启动后访问http://localhost:9000    查看
